# Lumière Skin - Marketing Landing Page

Standalone marketing site for Lumière Skincare brand. Pre-launch waitlist + brand storytelling.

## Features

- ✨ Hero section with waitlist CTA
- 🧴 Product showcase (static)
- 🔬 Science/Ingredients section
- 💬 Testimonials
- 📧 Email waitlist capture
- 📱 Fully responsive

## Getting Started

```bash
# Install dependencies
pnpm install

# Run dev server
pnpm dev

# Build for production
pnpm build
```

## Deployment

Configured for static export. Deploy to Vercel, Netlify, or any static host:

```bash
pnpm build  # Outputs to /dist
```

## Environment Variables (optional)

Create `.env.local`:

```env
RESEND_API_KEY=your_key_here
```

## Project Structure

```
app/              # Next.js app router
components/       # React components
  sections/       # Page sections
  ui/             # UI components
lib/              # Utilities
public/           # Static assets
```
